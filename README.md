## head

- A CSS rule consists of a selector and a declaration block. The selector points to the HTML element you want to style. The declaration block (inside curly braces {}) contains one or more declarations, which are the actual styles to be applied.

- A declaration is a property-value pair (e.g., property: value;) that defines a specific style.
